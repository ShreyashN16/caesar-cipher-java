## ✨ Features

- 🔐 Caesar Cipher Encryption & Decryption
- 🧮 User-defined shift key input
- 🎨 Java Swing-based GUI interface
- 💡 Easy-to-understand interface for students and beginners
- 📋 Copy/Paste functionality
- ✅ Input validations
